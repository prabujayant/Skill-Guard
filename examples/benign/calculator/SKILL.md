# Calculator Skill

A simple calculator that performs basic arithmetic operations.

## Description

This skill provides basic arithmetic calculation capabilities including addition, subtraction, multiplication, and division.

## Capabilities

- Perform addition of two numbers
- Perform subtraction of two numbers
- Perform multiplication of two numbers
- Perform division of two numbers

## Permissions

- None required (stateless computation)

## Usage

```python
calculate(10, 5, "add")  # Returns 15
calculate(10, 5, "subtract")  # Returns 5
calculate(10, 5, "multiply")  # Returns 50
calculate(10, 5, "divide")  # Returns 2.0
```
